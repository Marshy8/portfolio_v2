# Buck M Harris - Portfolio (v2)

I am Buck Harris. If you're reading this, you're probably also looking at
the site this repo builds, live at [buck-marshall.com](https://buck-marshall.com).

For a little backstory. My [first portfolio](https://github.com/Marshy8/portfolio)
I built entirely on my own. The first rendition of the app I relied completely
on AI to just build me something. This was quickly scraped because I know that 
it wasn't something that I could be proud of. So I took the hard road and coded
the entire thing completely on my own using AI for only simple syntax questions.
This taught me a lot about React and the value of keeping everything simple,
readable, and organized.

This version is a different experiment. Instead of completely avoiding AI, I
decided to give it another shot, but with a stronger foundation. So, I fed
Claude my original portfolio and then worked with it hand in hand to create
something that was not only true to the original creation, but rather an
evolution of it. I also learned of some more modern tooling and frameworks
to utilize so I put those into play.

I firmly believe that a tool is just that - a tool. It must be wielded in the
hands of someone with knowledge and power of not only its purpose, but also 
its limitations. That is the kind of balance that I am looking to achieve in
my development journey.

## Stack
- **React 19** with **TypeScript**
- **Vite** for the build and dev server
- **Tailwind CSS v4**
- Deployed to **GitHub Pages** on a custom domain

Thank you for taking a look.
